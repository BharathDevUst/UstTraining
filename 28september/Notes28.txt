# Spring Security

using JWT (JSON Web Token) for authentication vs authorization

**authentication is the first step**

- verifies user identity

**authorization comes after authentication**

- verifies your access

**Two types of authenticcation**

- JWT (Token based)
- Session based 

JSON web Token(JWT)
securely transfer information over the web(between two parties).

A JWT represents a string in the following format:
`[header].[payload].[signature]`

- header
- payload
- signature
